## 🛒 Lightweight POS & Inventory System
A lightweight, 100% offline-first retail counter and stock control system.
------------------------------
## 🌟 Core Highlights

* ⚡ Offline-First: Born to works without internet connection.
* 🔒 Data Privacy: Zero backend servers. All data stay in device.
* 📐 Optimized User Experiance: Large touch targets, high visual contrast, landscape Design. preventing misclicks in fast operation.
* 🌍 Bilingual: Supports English (EN) and 简体中文 (zh-CN).

------------------------------
## 🛠️ Key Features## 1. Counter Ordering

* Product Package : Pack several products into a package.
* Menu Groupping : Groupped display easier to navigate product.
* Order Archives: View past sales history and track order breakdowns.

## 2. Recipe-Driven Stock Control

* Product Availability: Availability calculated live from raw material stock.
* Automated Costing: Product cost calculated from ingredients, no manual input.
* Proportional Deduction: Checkout instantly deducts raw material stock.

## 3. Data Integration

* Data Export: Export sales reports via Excel and CSV. 

------------------------------
## 🚀 Tech Stack

* Core: Vue 3 + TypeScript
* Styling: Tailwind CSS
* Database: IndexedDB
* Offline Engine: vite-plugin-pwa
